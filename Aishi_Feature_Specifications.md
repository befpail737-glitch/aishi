# Aishi Distributor 英文网站 - 功能规格文档

## 文档概述

本文档详细描述Aishi英文网站各个页面和功能模块的具体规格，为开发和设计提供详细指导。

## 1. 首页 (Homepage)

### 1.1 英雄横幅 (Hero Banner)
- **功能**: 强调"全球领先的铝电解电容制造商合作伙伴"的定位
- **设计要求**: 视觉冲击力强，包含高质量图片或动画
- **内容要求**: 
  - 主标题: "全球领先的Aishi电容器授权代理商"
  - 副标题: "为全球客户提供原厂级技术支持与优质服务"
  - CTA按钮: "获取技术咨询"、"查询产品库存"
- **SEO要求**: 包含核心关键词"Aishi代理"、"艾华代理商"

### 1.2 核心优势 (Why Choose Us)
- **功能**: 展示作为艾华一级代理的竞争力
- **内容**:
  - 原厂直供保障
  - 快速交期承诺 (如: 7-15天)
  - FAE技术支持
  - 库存充足
- **设计**: 图标+简短描述形式，共4个优势点
- **SEO要求**: 每个优势点融入相关关键词

### 1.3 核心产品领域 (Core Product Areas)
- **功能**: 按类别展示产品，引导用户深入浏览
- **内容**:
  - 引线型铝电解电容 (Radial)
  - 贴片型铝电解电容 (SMD/V-Chip)
  - 牛角/基板自立型 (Snap-in)
  - 螺栓型 (Screw)
  - 固态高分子电容 (Solid Polymer)
  - 薄膜电容 (Film)
- **交互**: 每个类别包含链接到对应产品分类页
- **设计**: 卡片式布局，含SVG图标

### 1.4 解决方案与应用 (Solutions & Applications)
- **功能**: 展示在不同领域的解决方案
- **内容**:
  - LED照明驱动电源方案
  - 光伏逆变器/储能方案
  - PD快充/适配器方案
  - 工业控制方案
  - 新能源汽车方案
- **设计**: 每个解决方案包含缩略图、标题和简短描述
- **交互**: 支持轮播展示

### 1.5 最新动态 (Latest News)
- **功能**: 展示Aishi新品发布或市场动态
- **内容**: 显示最新的3-5篇文章标题和摘要
- **交互**: 点击可跳转到详细页面
- **自动更新**: 从新闻中心自动提取最新内容

### 1.6 最终行动号召 (Final CTA)
- **功能**: 底部引导用户获取服务
- **内容**:
  - 获取最新报价
  - 申请免费样品
  - 联系FAE工程师
- **设计**: 醒目的按钮，突出显示

## 2. 产品中心 (Product Center)

### 2.1 产品分类列表页
- **URL结构**: /products/[category]
- **功能**: 按6大分类展示Aishi电容器
- **分类**:
  - 引线型铝电解电容 (Radial Electrolytic Capacitors)
  - 贴片型铝电解电容 (SMD Electrolytic Capacitors)
  - 牛角/基板自立型 (Snap-in Electrolytic Capacitors)
  - 螺栓型电容 (Screw Terminal Capacitors)
  - 固态高分子电容 (Solid Polymer Capacitors)
  - 薄膜电容 (Film Capacitors)

### 2.2 产品分类详情页
- **功能**: 每个分类下的详细介绍页
- **内容结构**:
  - H1标题: [产品分类名称] + "Aishi代理"
  - 200-300字原创描述，融入关键词如"Aishi铝电解电容选型"、"艾华高压电容"
  - 产品介绍: 详尽中文介绍、关键参数、典型系列、应用领域及优势
  - 型号表格: 包含系列名、额定电压、容量范围、工作温度、耐久性、尺寸、纹波电流等
  - 选型指南入口: 链接到《如何根据纹波电流选择合适的Aishi电容？》
  - 代理商点评: FAE对产品的应用建议
  - 替代与配套: 提供替代方案和配套料号信息
  - FAQ模块: 3-5个常见问题解答
  - 多样化CTA: "下载规格书"、"申请免费样品"、"查询库存"、"咨询FAE"

### 2.3 型号表格功能
- **筛选功能**: 每个表格列都是筛选项
  - 电压筛选
  - 容量筛选
  - 温度范围筛选
  - 尺寸筛选
  - 纹波电流筛选
- **数据来源**: 基于Aishi官网真实数据
- **Schema标记**: JSON-LD格式的Product Schema

### 2.4 产品详情页
- **URL结构**: /products/[category]/[series]
- **功能**: 具体产品系列的详细信息
- **内容**:
  - 详细技术参数
  - 性能曲线图
  - 尺寸图
  - 应用电路
  - 替代方案推荐

## 3. 解决方案 (Solutions)

### 3.1 解决方案分类页
- **URL结构**: /solutions/[application]
- **内容**: 5大行业解决方案
- **页面结构**:
  - 方案框图 (Block Diagram with SVG or image with Alt text)
  - 核心优势 (耐高纹波、耐高温等)
  - 推荐物料清单 (BOM List，链接到对应电容系列)
  - 应用场景描述

### 3.2 解决方案详情页
- **内容长度**: 不少于800字
- **内容结构**:
  - 解决方案概述
  - 技术优势
  - 推荐产品
  - 应用案例
  - 技术细节
  - 相关资源链接

## 4. 技术支持 (Technical Support)

### 4.1 技术支持分类
- **分类**: 
  - 选型指南 (Selection Guides)
  - 应用笔记 (Application Notes)
  - 失效分析 (Failure Analysis)

- **标签**: 低ESR, 高纹波, 长寿命, LED驱动, PD快充, 变频器等

### 4.3 内容关联功能
- **上下文链接**: 提到特定系列时自动链接到产品页
- **相关文章**: 自动推荐3-5篇相关技术文章
- **作者简介**: 为FAE创建独立页面

### 4.4 技术文章详情页
- **内容长度**: 不少于800字
- **内容要求**: 解决工程师实际痛点
- **组件**: 
  - 详细技术说明
  - 图表和数据
  - 实际应用案例
  - 相关产品推荐
  - 底部CTA: "联系分销商"

## 8. 技术功能需求

### 8.1 SEO功能
- **URL结构**: 静态URL，结构清晰
- **面包屑导航**: 所有页面包含面包屑
- **Meta标签**: 优化所有页面的Meta标签
- **图片Alt属性**: 所有图片包含关键词的Alt属性
- **XML网站地图**: 自动生成sitemap.xml

### 8.2 内部链接策略
- **智能链接**: 技术文章中的技术术语链接到产品页
- **相关内容**: 产品页底部链接到技术文章
- **导航链接**: 清晰的页面间导航

### 8.3 响应式设计
- **适配设备**: 桌面、平板、手机
- **用户体验**: 不同设备下保持良好用户体验

### 8.4 性能优化
- **加载速度**: 优化图片和资源加载
- **缓存策略**: 合理的资源缓存
