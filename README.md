# Aishi Distributor Website

This is the official website for LiTong Group as an authorized distributor of Aishi (Hua Wei) electronic components. The website focuses on providing information about Aishi capacitors including radial, SMD, snap-in, screw terminal, solid polymer, and film capacitors.

## Site Structure

- **Home**: Main landing page with overview of services and products
- **Products**: Comprehensive product catalog with 6 main categories:
  - Radial Electrolytic Capacitors
  - SMD Electrolytic Capacitors
  - Snap-in Electrolytic Capacitors
  - Screw Terminal Capacitors
  - Solid Polymer Capacitors
  - Film Capacitors
- **Solutions**: Industry-specific applications and solutions:
  - LED Lighting Power Supply Solutions
  - Solar Inverter & Energy Storage Solutions
  - PD Fast Charging Solutions
  - Industrial Drive Solutions
  - EV OBC & BMS Solutions
- **Support**: Technical resources and documentation:
  - Selection Guides
  - Application Notes
  - Failure Analysis
  - FAQ
- **News**: Company updates and industry insights
- **About**: Company information and certifications
- **Contact**: Global sales network and contact form

## Technical Details

- Built with HTML5, CSS3, and JavaScript
- Responsive design for all device sizes
- SEO optimized with structured data
- Fast loading and optimized performance
- Cross-browser compatible
