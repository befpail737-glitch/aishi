# Aishi Distributor 英文网站 - 技术需求文档

## 1. 项目概述

### 1.1 项目名称
Aishi Distributor 英文网站

### 1.2 技术栈
- **前端**: HTML5, CSS3
- **部署平台**: Cloudflare Pages
- **构建工具**: 无特定要求，支持静态文件部署

### 1.3 域名
- **主域名**: aishi.elec-distributor.com

## 2. 前端技术需求

### 2.1 HTML规范
- **文档类型**: HTML5
- **字符编码**: UTF-8
- **语言声明**: `<html lang="en">`
- **响应式设计**: viewport meta标签必需
- **语义化标签**: 使用header, nav, main, section, article, aside, footer等语义化标签
- **可访问性**: 正确使用ARIA属性和role标签

### 2.2 CSS规范
- **样式标准**: CSS3
- **前缀处理**: 使用autoprefixer确保浏览器兼容性
- **响应式单位**: 使用rem、em、%、vw、vh等响应式单位
- **代码组织**: 按功能模块组织CSS代码
- **性能优化**: 避免过度嵌套，减少重复样式

### 2.3 浏览器兼容性
- **支持范围**:
  - Chrome 60+
  - Firefox 55+
  - Safari 12+
  - Edge 79+
  - IE不支持

### 2.4 性能要求
- **页面加载时间**: 所有页面在良好网络条件下3秒内加载完成
- **首字节时间**: 不超过1.5秒
- **Lighthouse性能分数**: 不低于85分
- **压缩**: 启用Gzip/Brotli压缩
- **缓存策略**: 合理设置资源缓存头

### 2.5 图片处理要求
- **格式**: 优先使用WebP格式，备用PNG/JPG
- **大小**: 根据使用场景提供多尺寸图片
- **懒加载**: 长页面中的图片启用懒加载
- **优化**: 所有图片在部署前进行压缩优化
- **Alt属性**: 所有图片必须包含描述性Alt属性

## 3. SEO技术规范

### 3.1 URL结构要求
- **静态URL**: 使用静态URL结构，避免参数化URL
- **URL层级**: 最多3层（域名/主分类/子分类/页面）
- **URL规范**:
  - 使用连字符(-)分隔单词，不使用下划线(_)
  - URL全部小写
  - 避免特殊字符
  - URL应具有描述性

### 3.2 Meta标签规范
- **Title标签**:
  - 长度控制在60字符以内
  - 包含核心关键词
  - 每页唯一
- **Meta Description**:
  - 长度控制在160字符以内
  - 包含核心关键词
  - 每页唯一且描述准确
- **Meta Keywords**:
  - 包含核心关键词: Aishi代理, 艾华代理商, Aishi电容代理, 艾华电解电容
  - 每页关键词略有不同以匹配内容

### 3.3 结构化数据标记
- **Product Schema**: 产品页面使用JSON-LD格式的Product Schema
- **Article/News Schema**: 新闻页面使用Article或NewsArticle Schema
- **Organization Schema**: 在关于我们页面添加公司组织信息Schema
- **Breadcrumb Schema**: 在所有页面添加面包屑Schema

### 3.4 内部链接规范
- **锚文本**: 使用描述性锚文本，包含关键词
- **链接结构**: 从首页到任一页面最多3次点击
- **死链检测**: 确保没有死链
- **相关链接**: 页面底部添加相关内容链接

## 4. 响应式设计技术要求

### 4.1 断点设置
- **移动端**: 320px - 767px
- **平板端**: 768px - 1023px
- **桌面端**: 1024px+

### 4.2 响应式要素
- **流式布局**: 使用百分比和弹性单位
- **媒体查询**: CSS媒体查询适配不同设备
- **图片响应**: 使用srcset和picture元素
- **字体响应**: 使用相对单位和媒体查询调整字体大小

### 4.3 移动端优化
- **触摸目标**: 最小44x44px触摸区域
- **手势支持**: 支持滑动等常见手势
- **加载性能**: 移动端优先优化加载速度
- **横屏适配**: 横屏模式下页面正常显示
