# Aishi Distributor 英文网站 - UX/UI设计规范

## 1. 设计概述

### 1.1 设计目标
- 专业可靠的电子元件分销商形象
- 直观易用的产品浏览和搜索体验
- 技术文档易于查找和阅读
- 促进用户咨询和转化

### 1.2 设计原则
- **专业性**: 体现技术导向和专业服务
- **清晰性**: 信息层次分明，易于理解
- **易用性**: 符合工程师用户使用习惯
- **响应性**: 跨平台一致的用户体验
- **可访问性**: 支持不同能力用户的访问需求

## 2. 视觉识别 (Visual Identity)

### 2.1 色彩方案
- **主色调**:
  - 深蓝色: #1a365d (用于导航、标题)
  - 银灰色: #718096 (用于辅助文字)
- **强调色**:
  - 橙色: #e53e3e (用于CTA按钮、重要提示)
  - 绿色: #38a169 (用于成功状态、通过验证)
- **中性色**:
  - 白色: #ffffff (背景)
  - 浅灰: #f7fafc (次要背景)
  - 深灰: #2d3748 (深色文字)

### 2.2 字体规范
- **中文字体**: 
  - 主字体: "Microsoft YaHei", "PingFang SC", sans-serif
  - 备用字体: SimHei, sans-serif
- **英文字体**:
  - 主字体: "Inter", "Helvetica Neue", Arial, sans-serif
  - 备用字体: sans-serif
- **字体大小**:
  - 主标题: 2rem (32px)
  - 次标题: 1.5rem (24px)
  - 小标题: 1.25rem (20px)
  - 正文: 1rem (16px)
  - 小字: 0.875rem (14px)
  - 微型文字: 0.75rem (12px)

### 2.3 间距规范
- **基础单位**: 8px
- **微间距**: 4px (组件内微小元素间隔)
- **小间距**: 8px (按钮内间距、表单元素间隔)
- **中间距**: 16px (卡片间隔、文本段落间隔)
- **大间距**: 32px (页面模块间间隔)
- **超大间距**: 64px (页面主要区域间隔)

## 3. 页面布局规范

### 3.1 整体布局
- **最大宽度**: 1200px (桌面端)
- **移动端**: 100%宽度，响应式设计
- **边距**: 
  - 桌面端: 48px 左右边距
  - 移动端: 16px 左右边距

### 3.2 导航设计
- **页眉高度**: 80px
- **导航菜单**:
  - 水平导航（桌面端）
  - 折叠菜单（移动端）
- **Logo位置**: 左上角
- **联系信息**: 右上角（WhatsApp、WeChat图标）

### 3.3 内容区域布局
- **宽度**: 100%
- **内边距**: 48px 48px 0 48px（桌面端）
            16px 16px 0 16px（移动端）

### 3.4 页脚设计
- **高度**: 最小120px
- **内容**: 
  - 全球联系信息网格
  - 版权信息
  - 社交媒体链接

## 4. 组件设计规范

### 4.1 按钮 (Buttons)
- **主要按钮 (Primary)**:
  - 背景色: #e53e3e
  - 文字色: #ffffff
  - 圆角: 4px
  - 高度: 44px
  - 内边距: 0 24px
  - 字体: 1rem, Bold
- **次要按钮 (Secondary)**:
  - 背景色: transparent
  - 边框色: #e53e3e
  - 文字色: #e53e3e
  - 其他属性同主要按钮
- **幽灵按钮 (Ghost)**:
  - 背景色: transparent
  - 边框色: #a0aec0
  - 文字色: #4a5568

### 4.2 表单元素
- **输入框**:
  - 高度: 44px
  - 边框: 1px solid #e2e8f0
  - 圆角: 4px
  - 内边距: 0 12px
  - 聚焦状态: 边框色变为 #e53e3e
- **选择框**: 
  - 样式与输入框一致
  - 右侧添加下拉箭头
- **复选框/单选框**:
  - 尺寸: 18px x 18px
  - 边框: 2px solid #cbd5e0
  - 选中状态: 背景色 #e53e3e

### 4.3 卡片 (Cards)
- **通用卡片**:
  - 背景色: #ffffff
  - 边框: 1px solid #e2e8f0
  - 圆角: 8px
  - 阴影: 0 4px 6px rgba(0,0,0,0.05)
  - 内边距: 24px
- **产品卡片**:
  - 包含图片、标题、简短描述
  - 悬停效果: 轻微提升阴影

### 4.4 表格 (Tables)
- **表头**:
  - 背景色: #f7fafc
  - 文字色: #2d3748
  - 字体: Bold
  - 垂直对齐: 中间
- **表体**:
  - 交替行颜色: #ffffff 和 #f7fafc
  - 边框: 1px solid #e2e8f0
- **响应式**: 移动端水平滚动或堆叠显示

## 5. 页面特定设计规范

### 5.1 首页设计
- **英雄横幅**:
  - 高度: 500px（桌面端）, 300px（移动端）
  - 背景: 渐变或高质量工业/科技图片
  - 文字对比度: 满足AA级无障碍标准
- **核心优势模块**:
  - 4个卡片水平排列（桌面端）
  - 1个卡片堆叠排列（移动端）
  - 每个包含图标、标题和简短描述
- **产品分类展示**:
  - 6个分类图标水平滚动（移动端）
  - 3x2网格布局（桌面端）
- **解决方案展示**:
  - 水平滑块设计
  - 每个解决方案包含缩略图、标题、描述

### 5.2 产品中心设计
- **分类导航**:
  - 垂直侧边栏（桌面端）
  - 顶部标签页（移动端）
- **型号表格**:
  - 支持列筛选和排序
  - 每页显示20行数据
  - 分页组件
- **产品详情**:
  - 左侧产品图片，右侧详细信息
  - 技术参数表格清晰展示

### 5.3 技术支持页面
- **文章列表**:
  - 卡片式布局
  - 显示标题、摘要、标签、日期
- **文章详情**:
  - 主内容区 + 右侧边栏
  - 侧边栏包含相关文章、标签云
- **内容层次**:
  - 清晰的标题层级
  - 适当的技术图表嵌入

### 5.4 响应式设计标准
- **断点**:
  - 小屏幕: 320px - 767px
  - 中屏幕: 768px - 1023px
  - 大屏幕: 1024px+
- **移动优先**:
  - 首先设计移动端体验
  - 逐步增强至桌面端
- **触摸优化**:
  - 最小触摸目标: 44px x 44px
  - 足够的间距避免误触

## 6. 交互设计规范

### 6.1 加载状态
- **页面加载**: 全局加载动画
- **异步操作**: 微型加载指示器
- **骨架屏**: 内容加载前的占位符

### 6.2 反馈机制
- **成功状态**: 绿色提示，配合检查图标
- **错误状态**: 红色提示，配合错误图标
- **警告状态**: 橙色提示，配合警告图标

### 6.3 导航交互
- **面包屑**: 点击可返回上级页面
- **分页**: 当前页高亮显示
- **标签页**: 点击切换内容

## 7. 视觉元素规范

### 7.1 SVG图标规范
- **尺寸**: 24x24px (默认), 可按需缩放
- **颜色**: 继承文本颜色，支持CSS控制
- **风格**: 线性图标，2px描边宽度

### 7.2 产品分类SVG图标
- **引线型电容**: 电路符号表示径向电容
- **贴片型电容**: 表示SMD封装的矩形
- **牛角电容**: 表示大型螺栓型封装
- **固态电容**: 表示聚合物电容特性
- **薄膜电容**: 表示薄膜介质特性

### 7.3 背景装饰元素
- **电路板纹路**: 用于页眉页脚背景
- **波形图案**: 用于技术内容页面
- **电荷符号**: 用于点缀科技感

## 8. 可访问性规范

### 8.1 色彩对比度
- **AA级标准**: 正文文字与背景对比度 ≥ 4.5:1
- **AAA级标准**: 大号文字与背景对比度 ≥ 3:1

### 8.2 语义化标记
- **标题层级**: 正确使用h1-h6标签
- **ARIA标签**: 为交互元素添加适当的ARIA属性
- **键盘导航**: 支持Tab键导航所有交互元素

### 8.3 替代文本
- **图片Alt属性**: 所有装饰性和内容图片必须有描述性Alt文本
- **图标说明**: 图标需提供文字说明或title属性
